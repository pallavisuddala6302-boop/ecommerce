# PHP & MySQL Development Environment Setup

## 📌 Project Overview
This project demonstrates the setup of a local development environment for PHP and MySQL. It includes configuring a local server using XAMPP and setting up version control with Git and GitHub.

## 🎯 Objective
The objective of this task was to create a fully functional local server environment for web development and establish version control for managing project files efficiently.

## 🛠 Tools & Technologies Used
- PHP
- MySQL
- XAMPP (Apache Server)
- Visual Studio Code
- Git
- GitHub

## 🚀 Steps Completed
1. Installed and configured XAMPP.
2. Started Apache and MySQL services.
3. Verified server setup using `http://localhost`.
4. Installed and configured Visual Studio Code for PHP development.
5. Installed Git and initialized a local repository.
6. Created a GitHub repository and pushed the initial commit.
